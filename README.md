# 🤖 Chatbot Seatech - L'Assistant IA de l'École d'Ingénieurs de Toulon  

Après **plusieurs tentatives et défis techniques**, nous avons enfin réussi à créer **un chatbot performant dédié à Seatech**, l’école d’ingénieurs de l’Université de Toulon. 🎓  

Ce chatbot a pour mission d’aider **les étudiants, futurs étudiants et curieux** à trouver facilement **des réponses à leurs questions** sur Seatech : admissions, formations, vie étudiante, etc.

## 📥 Installation Locale  

Si vous souhaitez exécuter le chatbot sur votre propre machine, suivez ces étapes 👇  

```sh
# 1️⃣ Cloner le projet
git clone https://github.com/farah746/chatbot_seatech.git
cd chatbot_seatech

# 2️⃣ Créer et activer un environnement virtuel
python -m venv venv
source venv/bin/activate  # macOS/Linux  
venv\Scripts\activate      # Windows  

# 3️⃣ Installer les dépendances
pip install -r requirements.txt

# 4️⃣ Lancer l’application
streamlit run chatbot.py

⚙️ Technologies utilisées
Ce projet est basé sur des outils puissants et modernes :

🐍 Python - Langage principal du chatbot
⚡ Groq - Modèle IA pour générer les réponses
🎨 Streamlit - Interface utilisateur interactive
🏪 ChromaDB - Base de données vectorielle pour la recherche d’informations
📄 FAISS & LangChain - Optimisation de la gestion des connaissances
💡 Fonctionnalités du Chatbot
✅ Réponses intelligentes et précises 📚
✅ Accès rapide aux informations sur Seatech 🎓
✅ Base de données optimisée pour la recherche 🔎
